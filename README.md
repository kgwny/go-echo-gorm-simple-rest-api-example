# go-echo-gorm-simple-rest-api-example

## セットアップ

### golang のバージョンアップ（Mac/Homebrew）
```
brew upgrade go
```

### golang のバージョン確認
```
go version
```

### golang のバージョン確認結果
```
❯ go version
go version go1.24.4 darwin/arm64
```

### go mod 初期化
```
go mod init my-app
```

### echo のインストール
```
go get -u github.com/labstack/echo/v4
```

### gorm のインストール
```
go get -u gorm.io/gorm
```

### mysql ドライバーのインストール
```
go get -u gorm.io/driver/mysql 
```
